# Archived Simulink Reference Assets

This directory preserves the older Simulink-side CADS reference material that
is no longer part of the active runtime path.

Contents include:

- legacy FMUs
- the original `.slx` model
- MATLAB helper code
- the large `inputfile.mat` reference signal

The current repo runtime does not load assets from this directory. Active FMUs,
workflow inputs, and generated test data live under the main `fmu/`, `data/`,
and `workflows/` paths instead.
